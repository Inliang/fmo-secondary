## 2. 配色方案

### 暗色主题 (默认)

| 用途 | 色值 | 说明 |
|------|------|------|
| 背景 | `#0D1117` | 深色基底 |
| 面板背景 | `#161B22` | 卡片面板 |
| 主色调 | `#00D9FF` | 冷蓝，信息/连接 |
| 强调色 | `#FFB000` | 琥珀，警告/高亮 |
| 危险色 | `#FF3E5C` | 品红，断开/错误 |
| 文字主 | `#E6EDF3` | 浅灰白 |
| 文字次 | `#8B949E` | 次级灰 |
| 网格线 | `#21262D` | 分割线 |

## 3. 组件设计

### 3.1 状态条 (Status Bar)

- 左：连接指示灯（绿/黄/红圆点）+ 状态文字
- 中：当前呼号 | 网格定位
- 右：FMO 地址 | 当前时间 | 设置按钮(齿轮)

### 3.2 SpeakingBar

- 当 `/events` 推送 `speaking_start` 时高亮显示
- 显示：讲话者呼号 + 地名反查结果 + VU 电平条
- 与对方通联次数统计
- 仅通联过 1 次显示「✦ 新朋友」徽章
- 5 秒无新 speaking 事件自动清空

### 3.3 频谱指示器（简化版）

- 基于 Audio WS 的 PCM 数据实时 FFT
- 显示为 16 段柱状频谱（参考 FmoDeck 的 SpeakingBar 迷你频谱）
- 仅音频连接激活时显示

### 3.4 QSO 列表

- 最近 8 条通联记录
- 每行：logId、对方呼号、网格、时间
- 自动滚动，最新在顶部
- 收到 `new_qso` 事件时插入并高亮 2 秒

## 4. 响应式断点

| 断点 | 布局变化 |
|------|---------|
| ≥ 1280px | 四象限标准布局 |
| 1024-1279px | 四象限缩小字号，面板内边距减小 |
| 768-1023px | 上下两行，每行两列 |
| < 768px | 单列堆叠，适合手机竖屏 |
